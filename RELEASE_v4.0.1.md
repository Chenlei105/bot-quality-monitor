# Release Notes v4.0.1

**发布日期**: 2026-03-27  
**类型**: 稳定版（Stable Release）

---

## 📋 概述

v4.0.1 是一个重要的维护更新，完成了 P0-P3 所有优先级任务，大幅提升了项目的可维护性、文档完整性和用户体验。

---

## ✨ 新功能

### 1. 完整的错误处理机制
- 飞书 API 失败自动重试（最多 3 次）
- 权限不足自动检测和用户通知
- 表格删除自动检测和配置清理
- 网络超时降级策略（连续 5 次后降为每小时采集）

### 2. 性能优化
- **增量采集**：已采集会话不重复扫描
- **批量写入**：攒够 10 条或 5 分钟后批量提交
- **限流保护**：单次最多 20 个会话，每秒最多 2 次 API 调用
- **幂等性保证**：collected-sessions.json 防止重复采集

### 3. Webhook 自动入库
- 每 10 分钟自动检查新消息
- 自动解析 JSON 数据
- 批量写入中央表格
- 幂等性保证（processed-webhook-messages.json）

### 4. 文档体系完善
- **5 分钟快速开始指南**（docs/QUICK_START.md）
- **故障排查完整指南**（docs/TROUBLESHOOTING.md）
- **数据隔离说明**（docs/DATA-ISOLATION.md）
- **授权模型文档**（docs/AUTHORIZATION.md）

## 📝 已知限制

1. **字段创建**：目前只创建核心字段，完整字段需要手动添加
2. **测试数据**：只写入 3 条演示数据，7 天完整数据待实现
3. **图表生成**：Plotly 图表代码已有，但需要完整数据才能展示
4. **跨企业采集**：依赖 Webhook，首次上报有最多 1 小时延迟
